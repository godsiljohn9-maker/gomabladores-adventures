# Gomabladores Adventures (Demo)

## Overview
**Gomabladores Adventures** is a **demo 2D action platformer** built with Python and Pygame.  
This demo showcases my work on:
- Character selection
- Mana charge system
- Basic combat mechanics
- Boss fight prototype
- Level progression concepts
  
- 🛠️ How It Was Built
Gomabladores Adventures was created using Python, Pygame, and fully custom pixel characters designed in Piskel. The project focuses on core gameplay systems and clean structure rather than large animation sets.

Core Components
- Pygame Engine — handled rendering, input, timing, audio, and game loops.
- Object‑Oriented Structure — player, enemies, UI, and levels organized into separate classes.
- Custom Physics & Collision — implemented gravity, jumping, hitboxes, and tile‑based collision logic.
- State Machine System — controlled transitions between title screen, character select, gameplay, pause, and boss fight.
- Simple Animation System — used basic frame‑by‑frame animations (no large sprite sheets) for idle, walking, and attacking.
- Combat & Mana Mechanics — built a basic attack system with cooldowns, mana charging, and special abilities.
- Boss Prototype — created an early boss with simple AI movement and attack patterns.
- Audio Integration — added sound effects and background music for feedback and atmosphere.

Art & Asset Creation
- Designed all characters and animations in Piskel, drawing each frame manually.
- Exported individual frames or small sequences directly into the project.
- Created simple backgrounds and tiles to test platforming and level flow.

Development Workflow
- Iterated frequently to refine movement, timing, and combat feel.
- Organized the project into folders (assets/, sprites/, levels/, scripts/) for clarity and maintainability.
- Used GitHub for version control and documentation of progress.

Tools Used
- Python 3.x
- Pygame
- Piskel (for all custom pixel art)
- VS Code / GitHub Desktop
